# **App Name**: NeonFolio

## Core Features:

- Visually Striking Project Gallery: Showcase a portfolio of projects with images or videos, project descriptions, and direct links to GitHub repositories.
- Interactive 'About Me' Section: Implement an 'About Me' section with a professional summary, a showcase of skills and a R3F 3D element, alongside coding profile links (icons) and a LinkedIn profile link.
- Integrated Contact Form: Provide a simple contact form for employers to get in touch.
- Dynamic Content Loading: Use React.js to dynamically load and render portfolio project data, about me information, and potentially fetch live data.
- Skills Proficiency Assessment: Given my uploaded resume or linked in profile the tool will rate and explain proficiency level using an LLM.

## Style Guidelines:

- Primary color: Neon blue (#39C0ED) for a futuristic and vibrant feel.
- Background color: Dark gray (#282C34) for a sleek, modern backdrop (dark scheme).
- Accent color: Purple (#B34BFF) for highlighting interactive elements.
- Body and headline font: 'Inter' for a modern, neutral aesthetic.
- Use simple line icons for navigation and skills representation.
- Full-screen layout with clear sections and a focus on visual hierarchy. Ensure a mobile-first responsive design.
- Subtle transition animations for page changes and interactive elements.
